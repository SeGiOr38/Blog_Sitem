SDÜ Bilgisayar Mühendisliği 3. sınıf öğrencisiyim. Kod yazmayı, yeni teknolojileri keşfetmeyi ve öğrendiklerimi paylaşmayı çok seviyorum. Bu site, hem dijital kimliğimi yansıtmak hem de projelerimi sergilemek için oluşturduğum kişisel alanım.

Nelerle Uğraşıyorum?

  • React ile modern ve kullanıcı dostu arayüzler geliştiriyorum
  • Node.js & Express ile backend servisleri tasarlıyorum
  • MongoDB ve SQL ile veri tabanı yönetimi ve modelleme
  • Kimlik doğrulama, güvenlik ve test süreçlerine odaklanıyorum
  • Docker ve basit DevOps süreçleriyle deployment çalışmaları yapıyorum

Projelerim

  • JavaScript ile interaktif bir hesap makinesi
  • Unity ile temel bir 2D platform oyunu
  • React Native ile mobil uygulama prototipi
  • REST API ile CRUD uygulaması geliştirdim

Becerilerim

  • Veritabanı - MySQL
  • Web Programlama - HTML, JavaScript, CSS, React
  • Mobil Uygulama - React Native, Flutter
  • Oyun Geliştirme - Unity, C, C#

Hedeflerim

Yakın gelecekte tam zamanlı bir full-stack geliştirici olarak sektörde yer almak, ekip çalışmasına katkı sağlamak ve açık kaynak projelere destek vermek istiyorum.
